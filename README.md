# C Programming Journey

A structured, level-based collection of C programming exercises — built from the ground up, one concept at a time. Each level introduces a new layer of the language, with progressively more complex problems designed to reinforce core fundamentals.

---

## Learning Path

| Level | Topic | Problems | Concepts Covered |
|-------|-------|----------|-----------------|
| [Level 01](./Level_01/README.md) | Arithmetic & Digit Manipulation | 25 | Operators, modulo arithmetic, digit extraction, branchless logic |
| [Level 02](./Level_02/README.md) | Boolean & Comparison Logic | 10 / 14 | Relational operators, boolean expressions *(in progress)* |
| Level 03 | Conditionals | 14 | `if`, `else`, control flow *(coming soon)* |
| Level 04 | Loops | 14 | `goto`-based loops, iteration patterns *(coming soon)* |

---

## How to Compile & Run

All programs use standard C (`C99` compatible). Compile any file with:

```bash
gcc filename.c -o output
./output
```

**Example:**
```bash
gcc 01_Add_2_to_Given_Number.c -o add
./add
```

> Tested on GCC. No external libraries required — pure standard C.
